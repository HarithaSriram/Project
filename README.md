  <h3>Azure Services used</h3>
  ------------------------------------------------------------------------------<br>
  <h4>1. Static WebApps<br>2. QnA Maker<br>3. Azure Bot Service</h4>
  <p>In order to create these Question and Answer pairs, I used QnAMaker. My Web App bot was integrated with QnAMaker. After that, I hosted it with Static Web App Service.</p>
  ------------------------------------------------------------------------------<br>
  <h3>Azure QnA Maker</h3>
  <p>I chose Azure QnA because of the following services:<br>
  1.Let customers take care of themselves.<br>
  2.Building a more knowledgeable database.<br>
  3.Improving the experience of users of any kind.<br>
  4.Easy answers to easy questions. You can answer frequently asked questions anytime and anywhere with your Bot.<br>
  5.We'll have quicker access to information.<br>
  </p>
  <h3> Azure Bot Service</h3>
  <p>
  Benefits of deploying a chatbot using Azure bot services include:<br>
  1.Even after deployment, we provide round-the-clock support.<br>
  2.Multiple platforms are easily integrated.<br>
  3.Customers interact more with the brand, increasing loyalty.<br>
  4.Superior cognitive features are taking customer engagement to an entirely new level.<br>
  </p>
  <h3>AZURE STATIC WEB APP</h3>
  <p>
    Static Web Apps from Azure are modern web apps that provide streamlined full-stack development, including source code and global high availability.<br>
    Here are the factors that led me to choose Azure Web Apps:<br>
    1.Streamlined management together with customized domain configuration and authentication and authorization<br>
    2.Seamless developer experience and CI/CD<br>
    3.Tailored developer journey that consists of a Visual Studio Code extension<br>
    4.Managed international availability for static content<br>
    5.Dynamic scale for serverless APIs<br>
  </p>
